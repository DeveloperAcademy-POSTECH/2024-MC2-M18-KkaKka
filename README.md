## :pushpin: Features

- **상대방과 연결하기**: 랜덤 코드를 통해 상대방의 기기와 쉽게 연결이 가능합니다.
- **메시지 보내기**: 미리 설정한 메세지 중에서 랜덤 혹은 정한 메시지를 상대방에게 전송할 수 있습니다.
- **푸시 알림**: 상대방에게 메시지가 오게 되면 푸시알림을 통해 확인이 가능합니다.
- **위젯을 통한 문구 받기**: 위젯을 통해 간단하게 상대방에게 랜덤 메시지를 보낼 수 있습니다.
